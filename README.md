## Описание

QuizLab - это Android-приложение для создания интерактивных викторин с возможностью:
- Создания и редактирования викторин
- Добавления вопросов вручную
- Автоматической генерации вопросов из контактов телефона
- Прохождения викторин с подсчетом результатов

## Технологии

- **Язык**: Kotlin 2.0.21
- **UI**: Jetpack Compose + Material 3
- **Архитектура**: MVVM + Repository Pattern
- **База данных**: Room Persistence Library 2.6.1
- **Навигация**: Navigation Compose 2.8.5
- **Асинхронность**: Kotlin Coroutines 1.9.0

## Требования

- Android 12 (API 31) и выше
- Разрешение READ_CONTACTS для импорта контактов

## Запуск проекта

1. Клонируйте репозиторий
2. Откройте проект в Android Studio
3. Синхронизируйте Gradle
4. Запустите на эмуляторе или физическом устройстве
